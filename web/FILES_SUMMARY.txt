ENVIRONMENT VARIABLES NEEDED
================================================================================

REQUIRED:
  DATABASE_URL                  - PostgreSQL connection string
  NEXTAUTH_URL                  - App base URL (http://localhost:3000)
  NEXTAUTH_SECRET               - Random secret (32+ chars)

STORAGE:
  S3_ENDPOINT                   - MinIO endpoint (http://minio:9000)
  S3_BUCKET                     - Bucket name (nodefleet)
  AWS_ACCESS_KEY_ID             - MinIO access key
  AWS_SECRET_ACCESS_KEY         - MinIO secret key

REDIS:
  REDIS_URL                     - Redis connection (redis://localhost:6379)

STRIPE:
  STRIPE_SECRET_KEY             - Stripe secret key
  STRIPE_PUBLISHABLE_KEY        - Stripe public key
  STRIPE_PRICE_PRO_MONTHLY      - Price ID for Pro plan
  STRIPE_PRICE_TEAM_MONTHLY     - Price ID for Team plan

OPTIONAL:
  NODE_ENV                      - development|production
  SMTP_HOST/PORT/USER/PASSWORD  - Email configuration
  SENTRY_DSN                    - Error tracking

================================================================================
GETTING STARTED
================================================================================

1. Install dependencies:
   cd /sessions/optimistic-pensive-pasteur/mnt/outputs/nodefleet/web
   npm install

2. Copy environment file:
   cp .env.example .env.local
   (Fill in required variables)

3. Generate database migrations:
   npm run db:generate

4. Push schema to database:
   npm run db:push

5. Run development server:
   npm run dev

6. Open browser:
   http://localhost:3000

7. Build for production:
   npm run build
   npm start
